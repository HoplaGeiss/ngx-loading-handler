ngx-loading-handler

Signals-based mutation state for Angular. One line replaces twenty.

GitHub npm

Basic usage

A single injectCommand() call manages loading, result, and error state automatically.

saveCommand = injectCommand(
  () => this.orderService.save(order)
);
@if (saveCommand.hasExecuted() && !saveCommand.isLoading()) {
✓ Saved — got ID {{ saveCommand.result()?.id }}
}

Error handling

hasError() and error() signals are set automatically on failure. No try/catch required.

failCommand = injectCommand(
  () => this.service.flaky(),
  { onError: (err) => toast.error(err.message) }
);
@if (errorCommand.hasError()) {
✗ {{ $any(errorCommand.error())?.message }}
}

Loading bar

Drop <ngx-loading-bar> once at the top of your shell. It shows a fixed progress bar while any command is loading. 

<!-- app.html -->
<ngx-loading-bar [command]="pageLoadCommand" />
